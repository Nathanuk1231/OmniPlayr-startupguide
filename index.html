Designed for extensibility.

A media player that doesn't lock you in. Install the sources you need, skip the ones you don't.

🔌
Plugin Architecture

Install only the sources you need. Each plugin is self-contained and independently managed.

🎧
Unified Playback

One consistent interface across all plugins — no jumping between apps or tabs.

📦
Self-Hosted via Docker

Your data, your server. Deploy with a single Docker Compose command.

🌐
Web UI Anywhere

Access the interface from any device on your network via browser — no client installs.

Quick Start.

Docker and Python 3.12+ required. Three commands and you're live.

terminal 
$ git clone https://github.com/OmniPlayr/OmniPlayr-server.git
$ cd OmniPlayr-server
$ python3 setup.py

# Then open your browser at:
→ http://localhost:3000

// manual start
docker compose 
$ docker compose up -d
// project layout

Structure.

Clean separation of concerns from the start.

OmniPlayr/
  ├── frontend/       Web UI
  ├── backend/        API server
  ├── setup/          Setup wizard (web-based)
  ├── db/             PostgreSQL data (auto-created)
  ├── docker-compose.yml
  ├── Dockerfile
  └── setup.py        Entry point
